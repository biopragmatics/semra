Artifacts
=========

Raw Mapping Database
--------------------

The raw mapping database can be reconstructed with the following command:

.. code-block:: console

    $ semra build

Domain-specific Processed Mapping Databases
-------------------------------------------

The domain-specific processed mapping databases and meta-landscape analysis can be
reconstructed with the following command:

.. code-block:: console

    $ semra landscape
